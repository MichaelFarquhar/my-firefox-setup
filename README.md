## Addons
- [Ublock Origin](https://addons.mozilla.org/en-CA/firefox/addon/ublock-origin/) - Ad and tracking blocker.
- [Bitwarden](https://addons.mozilla.org/en-CA/firefox/addon/bitwarden-password-manager/) - Password manager of choice.
- [I Still Dont Care About Cookies](https://addons.mozilla.org/en-CA/firefox/addon/istilldontcareaboutcookies/) - Removes the annoying cookies popups on websites. A fork of the orginal addon before it was purchased by Avast. 
- [Bypass Paywalls Clean](https://github.com/iamadamdev/bypass-paywalls-chrome) - Remove paywalls on a variety of sites. Needs to be installed separately due to Firefox removing it.
- [Allow Right Click](https://addons.mozilla.org/en-CA/firefox/addon/re-enable-right-click/) - Toggles right click on sites that block the feature.
- [HLS Downloader](https://addons.mozilla.org/en-CA/firefox/addon/hls-downloader/) - Video downloader for HLS streamed videos.
- [LocalCDN](https://addons.mozilla.org/en-CA/firefox/addon/localcdn-fork-of-decentraleyes/) - Protects agaisnt tracking through CDN's
- [ClearURLs](https://addons.mozilla.org/en-CA/firefox/addon/clearurls/) - Removes tracking elements from URLs.
- [Reddit Enhancement Suite](https://addons.mozilla.org/en-CA/firefox/addon/reddit-enhancement-suite/) - Makes reddit better.
- [Renewed Tab](https://addons.mozilla.org/en-CA/firefox/addon/renewed-tab/) - Customizable new tab with widgets.
- [Return Youtube Dislike](https://addons.mozilla.org/en-CA/firefox/addon/return-youtube-dislikes/) - Bringing back a feature that never should have been removed.
- [Search By Image](https://addons.mozilla.org/en-CA/firefox/addon/search_by_image/) - Reverse search with a variety of different services.
- [SponsorBlock](https://addons.mozilla.org/en-CA/firefox/addon/sponsorblock/) - Removes sponsors from youtube.
- [TWP - Translate Web Pages](https://addons.mozilla.org/en-CA/firefox/addon/traduzir-paginas-web/) - Translation addon of choice.
- [Unhook](https://addons.mozilla.org/en-CA/firefox/addon/youtube-recommended-videos/) - Removes features from youtube.
- [React Dev Tools](https://addons.mozilla.org/en-CA/firefox/addon/react-devtools/) - Dev tool.
- [Redux Dev Tools](https://addons.mozilla.org/en-CA/firefox/addon/reduxdevtools/) - Dev tool.
